#Repository Overview

This repository contains two main R Markdown files for the Sentence Memorability project:

p1_DataVis.Rmd: used for data preprocessing, validation, participant-level summarisation, and descriptive statistical visualizations.

p1_InferentialStats.Rmd: used for inferential statistical analysis, including repeated-measures ANOVA, paired t-tests, post-hoc comparisons, and report-ready summary values.
